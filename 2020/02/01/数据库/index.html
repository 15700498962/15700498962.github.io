  1. 启动服务

    net start 服务名：开启服务

    net stop 服务名 ：关闭服务

  2. 链接服务

    mysql -u 用户名 -p (-h -P（端口号）)

    show tables;

  3. 创建用户以及密码

    create user 用户名称 identified by ‘密码’；

    创建用户时分配权限

    grant 权限 on 范围 to 用户@ identified by ‘密码’；

  4. 选择

    select * from mysql.user \G; //格式化

  5. 服务器级别

    数据库（具体的某一个数据库）

    表

    列（字段）

  6. DESC 表名：查看表里数据的属性

  7. 授予权限

    grant 权限 on 范围 to 用户 [with grant option]

    [with grant option]代表被授权的用户可以将自己的权限授予别人

  8. 回收权限

    revoke 权限 on 范围 from 用户

  9. 刷新权限机制

    flush privileges;

  10. 修改密码

    1. 更改指定用户密码

      set password for c=password( ‘ ‘ );

   update mysql.user set password=password(' ') where user=" ";

语法

insert into 表名(字段) values(值列表)；

update 表名 set 字段1=值2 where 条件表达式

delete

注意：

  1. 如果针对表中所有字段添加数据，可省略字段列表
  2. 字段列表不是表中所有字段时，非空无默认值字段必须出现
  3. 值列表须与字段列表一一对应
  4. 对于字段列表中的允许为空字段，值可输入null或default代替用户未指定情况
  5. 对于字段列表中有默认值字段，值可使用defalut代替用户未指定情况

强调：

1.针对字符，日期，时间类型，数据值使用单引号。
