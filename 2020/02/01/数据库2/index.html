忘记密码怎么办？
  1. 打开mysql.exe和mysqld.exe所在的文件夹,复制路径地址
  2. 打开cmd命令提示符，进入上一步mysql.exe所在的文件夹。
  3. mysqld –skip-grant-tables
  4. 新打开一个cmd, 直接输入mysql
  5. use mysql ；
  6. select user,host,password from user ；
  7. update user set password=password(‘123456’) where user=’root’ and host=’localhost’;

auto_increment:自动递增

  1. 连续的时候会递增

  3. 删除后可以手动添加没有的数值，不手动添加的依旧会按照最大的数值自增
